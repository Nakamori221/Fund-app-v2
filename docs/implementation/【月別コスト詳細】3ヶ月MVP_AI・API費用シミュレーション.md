## 🛠️ 初期選定ツール一覧

### 必須ツール（MVP全期間）

```yaml
1. LLM API:
   選定理由: 情報抽出・レポート生成に必須
   選定モデル:
     - OpenAI GPT-5-nano: PUB収集・データ変換（最安）
     - OpenAI GPT-5: レポート生成（高品質）

2. データ管理基盤:
   選定理由: MVP開発・運用基盤
   選定ツール:
     - Google Workspace Business Standard

3. 外部データ収集（有料サービス）:
   選定理由: MVP期間は不要（月3案件なら手動検索が効率的）

   検討したが見送り:
     - Crunchbase Pro: $49-249/月（¥7,350-37,350/月）
       → 無料版 + プレスリリースで代替

     - SimilarWeb Starter: $199/月（¥29,850/月）
       → SaaS案件以外では不要、無料版で概算確認

     - LinkedIn API: $59-699/月（¥8,850-104,850/月）
       → Partner Program承認困難、手動検索の方が速い（5分/案件）

4. 補助ツール:
   選定理由: コスト抑制のため無料ツール活用
   選定ツール:
     - PDF処理: pdfplumber（無料ライブラリ）
     - スクレイピング: 手動コピペ
```

### 段階的追加ツール

```yaml
Month 1:
  - OpenAI GPT-5-nano（基本抽出）
  - OpenAI GPT-5（レポート生成テスト）

Month 2:
  - Claude Sonnet 4.5（複雑分析の検証）※オプション
  - Google Apps Script（無料、自動化）

Month 3:
  - プロンプトキャッシング活用（コスト削減）
  - Apify無料枠テスト（自動化検討）
```
