
総合ブラッシュアップ提案
## 【1】フォーマット２の改善

### 1-1. ステージ別テンプレート設計

現行のフォーマットは包括的ですが、投資ステージによって重要度が異なります。

```
【シード期】最重要項目（★★★）
- 課題の深刻度と市場機会
- 創業者の実行力とコミットメント
- プロダクト仮説の検証状況
- 初期トラクション（LOI、pilot顧客）
- チームの完全性

【アーリー期】最重要項目（★★★）
- PMF（Product-Market Fit）の証明
- ユニットエコノミクスの健全性
- GTM戦略の実行可能性
- 次回ラウンドまでのマイルストーン達成可能性
- 競合優位性の持続性

【レイター期】最重要項目（★★★）
- スケーラビリティの実証
- ガバナンス体制の成熟度
- エグジット戦略の具体性
- 財務の透明性と予測精度
- オペレーショナルエクセレンス
```

### 1-2. クリティカルパス分析の追加

投資判断に直結する「決定木」を明確化：

markdown

```markdown
## 【投資判断のクリティカルパス】

### Go/No-Go判定フロー
1. **Deal Breaker確認**（1つでもNoなら即却下）
   - 法的・規制上の致命的問題
   - 創業者の誠実性に関する重大な懸念
   - Cap Tableの修復不可能な歪み
   - 市場機会の消失

2. **必須条件確認**（全てYesが必要）
   - 解決する課題の実在性と深刻度
   - ソリューションの技術的実現可能性
   - チームの実行能力
   - 投資条件の妥当性

3. **競争力評価**（相対評価で上位30%以内）
   - 市場機会の大きさ
   - 競合優位性の強度
   - 期待リターンの魅力度
```

### 1-3. 項目の整理統合

重複や冗長性を削減：

**統合前 → 統合後**

- 「リスク」セクション + 各所のリスク言及 → **統合リスクマトリックス**（発生確率×影響度のマトリックス、各リスクの対策とオーナー明記）
- 「市場規模」+ 「なぜ今」の市場成長 → **市場機会分析**（TAM/SAM/SOM + 成長ドライバー + タイミング）
- 複数箇所の財務情報 → **統合財務ダッシュボード**（現状・計画・KPI・前提が一覧可能）
