# Fund Project

投資委員会資料作成とAIワークフロー管理プロジェクト

## 概要

このリポジトリは、投資ファンドの業務効率化を目的としたドキュメント管理システムです。
投資委員会資料のフォーマット、AIワークフローの設計、実装ガイドラインなどを含みます。

## プロジェクト構成

### 主要ドキュメント

- **投資委員会資料フォーマット**: 標準的な投資委員会資料のテンプレート
- **AIワークフロー設計**: AI活用による業務効率化の設計書
- **実装ガイド**: システム実装のためのステップバイステップガイド
- **評価レポート**: プロジェクトの進捗と評価

### ステップバイステップガイド

1. 指標定義辞書
2. セクター別マルチプルの構造と収集ルール
3. LP開示のマスキング規則
4. ダミー案件でのテスト実施

## 使い方

各ドキュメントは独立して参照可能です。
プロジェクトの段階に応じて、該当するステップのドキュメントを参照してください。
